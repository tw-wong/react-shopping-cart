# React-shopping-cart

## Project setup

```sh
# install and use Node 14
nvm install 14.17.1
nvm use v14.17.1

npm install
```

## Quick start

```sh
npm start
```

Then open http://localhost:3000/ to see the app.

## ESLint

```sh
# checking
npm run lint

# auto fix
npm run lint:fix
```

Note: Target for .js, jsx files.

## Prettier

```sh
# checking
npm run format

# auto fix
npm run format:fix
```

Note: Target for .html, .css, .json, .md files.
